# 🎮 Guild Bot - Imperia Of Titans

Telegram bot для гильдии **Imperia Of Titans** в игре **Rucoy Online**.

## ✨ Функции

### 1. 🔔 Автоматические уведомления
- Проверка сайта гильдии каждые 5 минут
- Уведомление о новых членах гильдии
- Уведомление об ушедших членах
- Отправка в установленный канал/тему чата

### 2. 📱 Команды для всех
- **`/online`** - Показать онлайн-членов
- **`/lvl`** - Топ-5 членов по уровню
- **`/start`** - Помощь и информация

### 3. ⚙️ Команды для админа
- **`/botguild`** - Установить канал для уведомлений

## 🌐 Интеграция
- **MongoDB** - облачное хранилище конфига и данных
- **Telegram Webhook** - бесплатный хостинг на Render
- **BeautifulSoup** - парсинг сайта гильдии
